* form태그 속성
1) action 속성 : form 태그 내부에 입력된 input태그 값을 전달할 서버 또는 페이지url 작성
2) method 속성 : 데이터 전송 방식을 지정하는 속성
method get : 주소를 통해 값은 전달하는 기본값
method post : http body부분의 내용을 담아 전달 (글자수 제한 o)

3) target 속성 : action으로 지정 된 경로를 현재/새 페이지 중 어디서 열지 선택
4) name 속성 : form태그에 이름을 지정하는 속성
