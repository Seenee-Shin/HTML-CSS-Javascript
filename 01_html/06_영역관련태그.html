영역 관련 태그

block형식
- 공간을 수직 분할하는 영역
- 수직선 상의 영역이 존재 [ㅡ]
- width, height 사용가능
inline 형식
-공간을 수평 분할 하는 영역 [ | ]

block, inline 차이점 : 줄바꿈

div태그 : 화면 영역을 나누는 용도

첫 번째 영역
두 번째 영역
세 번째 영역

block 형식 태그는 이미 존재하는 태그의 다음줄에 영역이 설정된다

span태그 : 줄바꾼이 일어나지 않고 옆으로 다음 영역 설정

첫 번째 영역 두 번째 영역 세 번째 영역

block , inline 차이점 2 : 영역 지정 방식

block : 사각형 박스 형태고 한 줄 전체를 영역으로 지정
inline : 내용(content) 단위로 영역을 지정
