목록관련 태그

li태그(List Item) : ol, ul태그 내부에서 사용되는 목록 내용 태그

ul태그(unordered List) : 순서가 없는 목록태그

  • Java
  • DB
  • JDBC
  • HTML5

ol태그(ordered List) : 순서가 있는 목록태그

type 속성 : 순서를 나타내는 형식을 변경하는 속성
ol태그 옆에 작성

start 속성 : 순서 시작을 지정하는 속성 (지정 순서부터 순차적으로)
reversed 속성 : 역순 (지정된 숫자의 역순으로 목록생성)

  1. Java
  2. DB
  3. JDBC
  4. HTML5
