고전 이벤트 모델

    요소가 가지고 있는 이벤트 속성에 이벤트 핸들러를 연결하는 방법
    인라인 모델과 다르게 script태그로만 작성 할 수 있음.

고전 이벤트 모델 복습

이벤트 발생 객체

    이벤트 발생 객체는 이벤트 핸들러의 매개변수로 이벤트 정보를 전달함
    (매개변수명 e, event )

고전 이벤트 모델의 문제점

    고전 이벤트 모델은 하나의 요소에 같은 이벤트 리스너에 대한 동작을 여러개 지정할 수 없다.
    -> 여러 동작 작성 시 마지막 이벤트만 수행
