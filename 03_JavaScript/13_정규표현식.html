정규 표현식(regular Expression)

 
    특정한 규칙을 가지는 문자열의 집합을 표현하는 언어
    
    정규표현식을 사용하여 입력된 문자열에 대한 조건검색, 치환 조건문을 간단히 처리 가능

정규 표현식 참고 사이트

  • 작성한 정규표현식을 그림으로 나타내주는 사이트
  • 정규표현식 테스트 사이트
  • MDN - 정규표현식 설명

정규 표현식 객체 생성 및 확인 메소드

    1.정규 표현식 객체 생성 방법

    const regExp = new regExp("정규표현식")
    const refExp = /정규표현식/
    / : 정규표현식의 리터럴 표기법

    2.문자열 일치여부 확인 메소드

      1) regExp.test(문자열)
        -문자열에서 정규표현식과 일치하는 패턴이 있는 경우 true

      2) regExp.exec(문자열)
        - 문자열에서 정규 표현식과 일치하는 패턴이 있는 경우, 처음 매칭되는 문자열을 반환
        - 없으면 null반환

정규 표현식의 메타문자

    문자열의 패턴을 나타내는 특수문자
    특수문자마다 지정된 특별한 뜻이 담겨있다.

    ^ (캐럿) : 문자열의 시작을 의미
    $ : 문자열의 끝 
    \w : word
    \d : degit 0~9 사이 숫자
    \s : space
    * : 0개 이상 == 0번 이상 반복
    ? : 0개 또는 1개 
    + : 1개이상 == 1번 이상 반복 
    . : 개행문자를 제외한 모든 단일 문자
    ex) a로 시작하는 문자열 : /^a/


문자열이 j또는 a로 시작하고 t 또는 x 로 끝나는지 확인

수량 관련 문자 

    a{5} : a가 5개 == aaaaa
    a{2,5} : a가 2개 이상 5개 이하
    a{2,} : a가 2개 이상
    a{,5} : 5개 이하

플래그 문자 

    정규짓 객체 뒤에 붙이는 문자 

    g: 문자열 전체를 검색 (전역비교)
    i: 대소문자를 가리지 않고 비교 
    m: multi line 여러줄의 검사를 수행
