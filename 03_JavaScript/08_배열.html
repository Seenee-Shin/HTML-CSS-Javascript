배열

    JAVA의 배열 : 같은 자료형 변수가 연속적으로 나열 되어있는것
    
    JS의 변수 : 값이 리터럴이 저장 될 때 까지 자료형 지정x 
    JS의 배열 : 자료형이 정해져 있지 않은 배열 
      -> JAVA의 List와 유사

    특징 
    자료형 제한이 없다
    길이 제한이 없다

배열 관련 함수

    (stack 관련 함수)
    push("값") : 배열 맨 뒤에 요소를 추가
    pop() : 배열 맨 뒤 요소를 꺼냄

    indexOf("값"): 일치하는 값을 가지는 요소의 인덱스를 반환. 없으면 -1반환
    lastIndexOf("값"): 일치하는 값을 가지는 요소를 뒤에서 부터 검색, 인덱스를 반환. 없으면 -1반환

    toString() : 배열의 모든 요소를 하나의 문자열로 반환alcu
    join("구분자") : 배열 내 모든 요소를 하나의 문자열로 반환

    sort([정렬기준이 작성된 함수]): 배열 내의 요소를 오름차순으로 정렬
    reverse() : 배열 정렬 순서를 반대
