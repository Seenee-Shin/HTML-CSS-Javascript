JS의 객체

    JS의 개체는 {}내의 key :Value가 모여있는 상태로 작성
    (Map 형식)

    { k:v , k:V , k:v }
    
    (참고)
    자바 스크립트 객체 모양으로 작성된 문자열 
    "{ k:v , k:V , k:v }" -> JSON (Javascript Object Notation 자바스크립트 객체 표기법)

    객체 생성 방법 
    1. {} (객체 리터럴 표기법)을 이용한 생성 방법 
    2. 셍성자 함수 + new연산자를 이용한 생성 방법

생성자 함수

    객체 생성시 호출하는 함수 
    -일반적인 함수와는 다른 new연산자를 이용하여 객체생성
