window 객체

    브라우저 창 자체를 나타내는 객체

    -window 객체는 자바 스크립트의 최상위 객체
    DOM, BOM으로 나뉘어짐

    DOM (document objecjt model) : document
    BOM (browser object model) : location, history, screen, navigator

    window 객체는 창 자체를 의미하므로 어디서든 접근이 가능하다. 
    그래서 window 객체 호출부를 생략할 수 있다.

window.setTiimeout(함수, 지연시간(ms))

함수 실행 시간 지연

window.setInterval(함수, 지연시간(ms))

    지정된 지연 시간마다 함수가 반복 수행
    문제점 : 중복되어서 사용가능, 무분별하게 여러번 호출 시 성능 저하
