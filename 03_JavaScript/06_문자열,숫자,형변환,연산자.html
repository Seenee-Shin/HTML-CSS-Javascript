문자열

      '',""를 리터럴 기호로 사용
      String 자바 스크립트 내장 객체 제공
      해당 객체에는 문자열을 위한 기본적인 함수들을 제공함

숫자, 숫자 관련 함수

      숫자 자료형 (Number)의 리터럴 표기법
      - 10(정수), 41.3(실수), infinity(무한), NaN(Not a Number)

      Math: 숫자 관련 함수를 제공하는 JS 내장 객체

문자열 -> 숫자형변환

      숫자로만 이루어진 문자열을 정수, 실수 형태로 변환하는 방법이 존재함 

      parseInt("숫자로된 문자열") -> 정수로 변환
      paresFloat ("숫자로된 문자열") -> 정수는 정수로, 실수는 실수로 변환
      Number ("숫자로된 문자열") -> 정수는 정수로, 실수는 실수로 변환

동등/ 동일 비교 연산자

      동등 비교 연산자 (==. !=)
      -> 자료형에 관계없이 값이 같으면  true반환
      ex) 1 == "1"

      동일 비교 연산자 (===)
      -> 자료형과 값이 같아야 true
