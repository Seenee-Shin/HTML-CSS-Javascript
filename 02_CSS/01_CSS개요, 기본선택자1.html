CSS 기본선택자

선택자란?
현재HTML 문서내에 특정 요소를 선택하는 문자.

선택된 요소에 원하는 스타일을 적용할 수 있다.

태그 선택자

현재 html문서 내에 같은 태그를 선택하는 선택자
[작성법]

스팬태그 내부에 작성된 글자 강조 문자 스팬태그 내부에 작성된 글자

ID 선택자 (#id속성값)

HTML 문서 내에 id 속성값이 일치하는 요소를 선택하는 선택자.
(중요) 페이지 내에 중복을 허용하지 않는다.
[작성법] css코드
(# == id)

class 선택자 (.class 속성값)

HTNL문서내의 같은 class 속성값을 가지는 태그를 모두 선택하는 선택자
[작성법]
.class속성값 {코드}

모든 요소 선택자

단독 또는 다른 선택자와 함께 사용 가능한 선택자

단독 : 현재 html문서 내 모든 요소 선택
다른 선택자와 함께 : 해당 선택자 내부, 형제, 부모등을 모두 선택
