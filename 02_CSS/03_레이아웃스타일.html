화면 배치 방법(display : block | inline | inline-block)

display 속성 : 요소가 화면에 어떻게 보여질지 지정하는 속성

block : 화면을 수직 분할 (한줄을 모두 차지)
-> width / height 지정 가능

inline : 화면을 수평 분할 (한 줄에 여러 inline 요소가 배치됨)
-> width / height 지정 불가능

inline-block : inline 수평 분할 특성 + block의 크기 지정 가능 특성

none : 기능은 유지하되 화면에 보이지 않음
