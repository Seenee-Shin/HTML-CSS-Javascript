배치 관련 스타일 (position)

    position 속성을 요소의 위치를 지정하는 css속성.

    position : relative;
    - 이 스타일이 지정된 요소가 상대적으로 나타낼 수 있는 다른 요소의 기준이 됨.
      (자기 자신이 기분이 되어 다른 요소의 위치를 배치함 )
    - top, bottom, left, right 속성을 이용할 수 있게됨
      (relative가 설정된 요소의 자식이 사용가능)

    position : absolute; (절대적)
    - 기본 요소의 배치 순서를 무시하고 지정된 절대 위치에 요소를 배치함

    position : fixed; (고정)
    - 브라우저 창을 기준으로 항상 고정된 위치에 요소를 배치함
