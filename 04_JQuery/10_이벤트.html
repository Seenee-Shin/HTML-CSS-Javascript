input 태그 관련 이벤트 

    focus : input태그가 선택되어 있는 경우

    blur : 아웃포커싱 
          ex)input 태그에 커서가 깜빡거리는 중 다른 곳을 클릭했을 때 
    
    change : input 태그의 값 또는 체크 여부가 변했을 때 
          1) checkbox, radio, select
          -> 값이 변하자 마자 change 이벤트 발생 

          2)text 관련 input타입 (text, email, tel , url, search, number)
          -> 입력과 동시에 값이 변했다고 인지하는 것이 아니라 포커스를 잃은 경우에 이전 값과 
          현재값이 다르면 change 이벤트 발생

    select : 요소 입력 영역에 블럭이 설정 된 경우
