시각적 효과를 위한 effect 메소드 

    요소에 에니메이션 효과를 만들기 위한 메소드 집합

    1.show() / hide()
    2. fadeIn () / fadeOut()
    3. slideDown() / slideUp()

    *effect 메소드 작성 방법
    1. $(‘선택자’).메소드명();
    2. $(‘선택자’).메소드명(speed);
    3. $(‘선택자’).메소드명(speed, easing, callback);
    speed : 실행속도(ms) / 숫자 or slow, fast
    easing : 변경되는 지점별 속도 / linear, swing 가능
    callback : 메소드 실행 후 실행할 함수

1. show() / hide()

    show () : 요소를 점점 커지게 함 
    show () : 요소가 점점 작아지며 화면에서 없어짐

2.fadeIn() fadeOut()

3. slideDown() / slideUp()

토글 (toggle)
      - 하나의 버튼에 두가지 상태를 지정할 수 있는 버튼 (스위치)

      toggle()메소드 : 메소드 수행시 요소의 display 상태에 따라서 자동으로 show, hide를 생성
