jQuery 선택자

    jQuery 선택자는

    $("선택자")   의 형태를 띄고 있으며

    "선택자"의 작성 방법은 CSS 선택자 방식을 따른다.

태그 선택자

    [JS]
    document.getElementsByTagName("태그명");


    [jQuery]
    $("태그명");

아이디 선택자

    첫 글자는 영어 대문자만.
    나머지 글자는 영어 소문자, 숫자, !, @, # 만 가능
    전체 글자 수는 8 ~ 20글자;

기본 속성 선택자

    기본 속성 선택자 : 선택자 [속성="속성명"]

    [JS]
    document.querySelector("기본 속성 선택자")  == 요소 1개
    document.querySelectorALL("기본 속성 선택자") == 모든 요소

    [JQ]
    $("기본속성 선택자") : 일치하는 모든 요소
